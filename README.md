# IPL Analytics Dashboard (2008–2026) | Power BI

## Project Overview

This project is an interactive IPL Analytics Dashboard developed using Power BI to analyze IPL matches from 2008 to 2026.

The dashboard provides insights into team performance, match outcomes, toss decisions, venue statistics, player achievements, and season-wise trends through interactive visualizations and filters.

---

## Dashboard Pages

### 1. IPL Overview Dashboard

Key Insights:

* Total Seasons
* Total Matches
* Toss Decision Analysis
* Matches Won by Team
* Total Matches by Season
* Matches by City
* Highest Team Score
* Highest Successful Chase
* Toss Winner vs Match Winner
* Top 5 Player of the Match Winners

### 2. Team Analysis Dashboard

Key Insights:

* Team Wins by Season
* Top 10 Venues by Matches Hosted
* Highest Winning Margins
* Team Performance Analysis
* Team and Season Filters
